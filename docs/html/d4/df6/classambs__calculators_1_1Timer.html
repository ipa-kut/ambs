Detailed Description

Start counting down timer from a parametrized value to 0 when triggered and set timed out unless interrupted.

Has standard control intetrface, a float interface with one output port and a Bool interface with two inputs and two outputs.

◆ getNextState()

TimerState getNextState ( TimerState  current_state)
private

Calculate what the next state should be, given current state and external inputs.

The state machine is as follows, always starting from DISABLED: DISABLED -> IDLE IFF ENABLE=True IDLE -> RUNNING IFF START=True RUNNING -> FINISHED IF STOP=True OR (elapsed >= duration) <ANY STATE> -> DISABLED IFF DISABLE=True

Parameters
current_stateThe state the timer is in now
Returns
The next state the timer should be in

All states can and should transition to DISABLED.
